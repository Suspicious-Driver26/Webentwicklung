# Task Bingo 
Ein Tool, bei dem Benutzer ein individuelles Bingo Board mit wöchentlichen Aufgaben erstellen und diese erledigen können, um zufällige Preise zu gewinnen. Die Boards werden wöchentlich oder monatlich mit einem Countdown Timmer zurückgesetzt.
## Seitenstruktur
- 5x5 Bingo Board mit aktuellen Aufgaben.
- Seitenliste mit Aufgabenliste (Hinzufügen/Entfernen von Aufgaben)
- Bereich für Preisanzeige.
- Countdown Timer mit Anzeige der Zeit bis zum nächsten Board Reset.
